# Node.js chat

It's very simple chat based on Node.js

## Getting started
```
git clone https://github.com/artem-solovev/nodejs-chat.git
```

```
npm install
```

```
n–æde index.js
```

### Prerequisities
* Git
* Node.js

## Built with
* Node.js
* Express
* Ejs
* Socket.io
